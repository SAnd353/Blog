Section 8: Iteration

  • Iteration: repeating portion of an algorithm
  • Iteration statements: statements that create iteration
  • Repeat until: repeating on a boolean condition
  • finding lost item example: start with one room, search, if not found, go to next room, repeat until you find the item

while (i < 3):
    print(i)
    i += 1

Section 10: Lists

  • Traversal: visiting each element in a list sequentially
  • Complete, partial, iterative (using a loop) traversal
  • len(), sort() (permanent), sorted() (temporary), reverse (reverses a list)
  • range(), min(), max(), sum()
list = [1, 5, 8, 10]
print(len(list))
print(min(list))
print(max(list))
print(sum(list))

Consolidated vocabulary

  • Iteration: repeating portion of an algorithm
  • Iteration statements: statements that create iteration
  • Repeat until: repeating on a boolean condition
  • Traversal: visiting each element in a list sequentially
  • Complete, partial, iterative (using a loop) traversal (definitions are self-explanatory beyond "iterative")
